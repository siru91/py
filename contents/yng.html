연애능력고사
감성과학기술센터(ESRC)의 의뢰로 진행한 작업.
기획서를 바탕으로 한 GUI 제작, Xml을 이용한 구현을 담당했다.
사용자에게 주관평가, 표정짓기, 제시된 표정 맞추기 등을 수행하게 하며
그 때의 표정과 심박수를 측정하여 각 항목들을 점수로 환산, 결과로 보여준다.

작업 방식 디자인
'연애능력고사'의 이름에 맞게 사랑스러운 분위기를 요청받았고, 분홍 톤을 주색으로 설정하여 진행했다.
내부 요소들은 부드러운 느낌을 주기 위해 둥근 모서리가 적용된 사각형으로 제작했다.
사용자가 수행할 작업에 대한 튜토리얼이 필요했는데, 벡터 애니메이션을 이용해 제작했다.

구현
Windows app 기반 XML을 이용하여 구현했다. 당시 XML에 대해서 지식이 거의 없었는데,
XML의 우수한 사용성 덕분에 대부분의 구현을 비교적 쉽게 해결할 수 있었다.

결과

Pros
UI의 디자인과 구현까지의 프로세스를 한번에 경험해볼 수 있는 기회였다.
XML에 대해서 더 관심을 갖게 되었으며 걱정했던 것 보다 재밌었다.

Cons
백엔드를 담당하신 분이 굉장한 고수셨는데, 작업 방법들을 좀 더 배웠으면 좋았을 것 같다.
